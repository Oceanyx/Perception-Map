This is a starter scaffold for the Perception Map Playground.

## How to run

1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`

This project uses Vite + React + Tailwind + React Flow + Dexie.

Files included are minimal components to get the playground running. Expand CanvasView to add lens styling, domain overlays, and link creation.
